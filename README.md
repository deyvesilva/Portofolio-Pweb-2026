# 🚀 Sejam bem-vindos ao meu Portfólio Web em Django

Este projeto consistiu no desenvolvimento de uma aplicação web utilizando o framework Django para gestão de um portfólio académico e profissional.

A aplicação permite armazenar, gerir e visualizar informação relevante sobre o percurso académico e profissional, incluindo projetos, tecnologias, competências, formação, experiência profissional e Trabalhos Finais de Curso (TFC).

Foram também integradas fontes externas de dados, nomeadamente ficheiros JSON e APIs da Universidade Lusófona, permitindo enriquecer o sistema com informação dinâmica e atualizada.
